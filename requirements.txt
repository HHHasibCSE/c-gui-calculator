# Requirements for building and running the C GUI Calculator project

# Core Dependencies (Required):
# gcc           - GNU Compiler Collection v7.0+ - Required to compile C source code
# pkg-config    - Build configuration tool v0.29+ - Manages compiler/linker flags for GTK
# libgtk-3-dev  - GTK+ 3 development files v3.22+ - Required for GUI applications in C

# Optional Dependencies:
# build-essential - Development tools package (Linux)
# git             - Version control system for cloning repository
# make            - Build automation tool (recommended)

# Platform-Specific Installation:
#
# Linux (Ubuntu/Debian):
#   sudo apt update
#   sudo apt install gcc pkg-config libgtk-3-dev build-essential
#
# Linux (Fedora/CentOS/RHEL):
#   sudo dnf install gcc pkgconf-pkg-config gtk3-devel
#
# Linux (Arch/Manjaro):
#   sudo pacman -S base-devel gtk3 pkg-config
#
# Windows (MSYS2 MinGW64):
#   pacman -S mingw-w64-x86_64-gcc mingw-w64-x86_64-pkg-config mingw-w64-x86_64-gtk3
#
# macOS (Homebrew):
#   brew install gcc pkg-config gtk+3
#   brew install --cask xquartz  # Required for GUI display

# Core Requirements:
gcc            # C compiler (v7.0+)
pkg-config     # Build configuration helper (v0.29+)
libgtk-3-dev   # GTK+ 3 development files (v3.22+)

# Platform Notes:
# - Windows: Use MSYS2 MinGW 64-bit terminal
# - macOS: XQuartz required for GTK+ GUI applications
# - See Requirements.md for detailed setup instructions
